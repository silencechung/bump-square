# Toolbar + 快捷鍵

**Frame** toggle — OFF = Hand mode(預設,純 pan,frame 只能 click 選);ON = Frame mode(拖空白畫 frame、拖 frame body 畫巢狀、**`Ctrl + 拖 frame body` 移動 frame** ← 唯一移動路徑)。
**frames 計數** + **↶ / ↷** — undo / redo 只動 board。

## 快捷鍵

| 鍵 | 動作 |
| --- | --- |
| `Esc` | 取消 placing → 收選取 → 退出 Frame → fit |
| `Space` 按住 | 暫時 pan |
| `F` / 雙擊空白 | fit overview |
| 雙擊 frame | 對焦到那個 frame |
| 滾輪 | 對游標位置 zoom |
| `Ctrl+Z` / `Ctrl+Shift+Z` | undo / redo |
| `Ctrl+C` `Ctrl+X` `Ctrl+V` | 複製 / 剪下 / 進入放置模式 |
| ``Ctrl+` `` | terminal panel |
