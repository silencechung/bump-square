# 畫布:Frame + 意圖

在截圖上畫 frame,每塊寫一句「**你想要什麼**」— 那就是給 agent 的意圖。

- 預設 **Hand mode**:純 pan(拖空白或 frame 都 pan canvas),frame 只能 click 選取
- 切到 **Frame mode** 才能新增 / 移動:拖曳空白畫 frame、拖曳 frame body 畫巢狀;**`Ctrl + 拖曳 frame body`** 是唯一移動 frame 的路徑
- 選中 frame 後出現 8 個 handle 可 resize(兩種模式都有)、`⧉` 複製含內部
- 框依**面積排 z-index**,小框在上面,可點進內層
- 選中框會在右側 Notes rail 畫一條虛線 leader line 指到它的編輯列

工具 / 快捷鍵完整列表在 toolbar 左上的 dot。
