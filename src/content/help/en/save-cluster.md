# Saves + path copy

**💾 Save** — persists the current board as a named JSON under `~/.bump-square/saves/`. Multiple saves coexist; switch between them.

**Copy path** — puts the latest save's absolute path on the clipboard, ready to paste to a downstream reader skill / dev agent / a friend who'll read the file.

This is bump-square's handoff — there is no "submit" button; what happens downstream to the intent is up to you.
