# Toolbar + shortcuts

**Frame** toggle — OFF = Hand mode (default, pure pan, frames are click-select only); ON = Frame mode (drag empty space to draw a frame, drag a frame body to draw nested, **`Ctrl + drag frame body` to move a frame** ← the only move path).
**frame count** + **↶ / ↷** — undo / redo, board only.

## Shortcuts

| Key | Action |
| --- | --- |
| `Esc` | cancel placing → clear selection → exit Frame mode → fit |
| `Space` (hold) | temporary pan |
| `F` / double-click empty | fit overview |
| double-click frame | focus on that frame |
| wheel | zoom around the cursor |
| `Ctrl+Z` / `Ctrl+Shift+Z` | undo / redo |
| `Ctrl+C` `Ctrl+X` `Ctrl+V` | copy / cut / enter placing mode |
| ``Ctrl+` `` | terminal panel |
